# Installation
Download from Release page and put into ASF/plugins

Tested for ASF-generic 5.1.1.1

# Commands
- gbphelp – Get help for this plugin
- gbpbuygame [Bots] \<AppID> \<SubID> – Buy game by AppID and SubID
- gbpbuybundle [Bots] \<BundleID> – Buy bundle by BundleID
- gbpbuyitem [Bots] \<AppID> \<ItemID> - Buy one item by AppID and ItemID
- gbpbuyitem [Bots] \<AppID> \<ItemID> \<Quantity> – Buy several items by AppID and ItemID
- gbpclearcart [Bots] – Clear Steam cart
